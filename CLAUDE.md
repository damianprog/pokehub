# PokeHub

PokeHub is an opinion-first, Letterboxd-style social platform for rating, reviewing, and listing Pokémon, layered with a decoupled daily pack-collection system that adds a return hook without gating any social features behind it.

## Commands

```bash
npm run dev        # Start dev server (Turbopack, outputs to .next/dev)
npm run build      # Production build (Turbopack by default)
npm run start      # Start production server
npm run lint       # Run ESLint directly (next lint is removed in v16)
```
